Variables con var en JavaScript

¿Qué es una variable?

Una variable es un contenedor que almacena un valor de datos. En JavaScript, var es una de las formas de declarar variables.

1. Declaración Básica de Variables

Podemos inicializar una variable con un valor:

var producto = "Semitas"; console.log(producto); // Resultado: Semitas

2. Reasignación de Variables

Las variables declaradas con var se pueden reasignar:

var producto = "Semitas"; producto = "Cafe"; console.log(producto); // Resultado: Cafe

3. Tipado Dinámico

JavaScript es un lenguaje de tipado dinámico, no necesitamos especificar el tipo de dato:

var producto = "Cafe"; // String producto = 20; // Number producto = true; // Boolean console.log(producto); // Resultado: true

4. Variables sin Valor Inicial

Podemos declarar variables sin asignar un valor inicial:

var disponible; // undefined disponible = true; // Asignamos valor después console.log(disponible); // Resultado: true

5. Declaración de Múltiples Variables

Podemos declarar múltiples variables en una sola línea:

var categoria = 'Panes', marca = 'La Popular', calificacion = 5; console.log(categoria, marca, calificacion);

6. Reglas de Nomenclatura

Ejemplos de nombres válidos e inválidos para variables:

// ❌ INCORRECTO - No puede empezar con número // var 99dias; // ✅ CORRECTO var dias99; var _usuario; var __usuario; var $precio;
Regla importante: Los nombres de variables no pueden comenzar con un número.

7. Convenciones de Nomenclatura

Diferentes estilos para nombrar variables con múltiples palabras:

// Camel Case (Recomendado en JavaScript) var nombreProducto; // Pascal Case var NombreProducto; // Snake Case var nombre_producto; // Sin separación (No recomendado) var nombreproducto;
Recomendación: En JavaScript se prefiere el camelCase para nombrar variables.

8. Ámbito (Scope) de var

Las variables var tienen ámbito de función y pueden ser redeclaradas:

function ejemploAmbito() { var x = 10; if (true) { var x = 20; // Esta x sobrescribe la anterior console.log("Dentro del if:", x); // 20 } console.log("Fuera del if:", x); // 20 }
Importante: var no respeta el ámbito de bloque, solo el de función.

Puntos Importantes

  • var se usa para declarar variables en JavaScript
  • Las variables pueden ser reasignadas
  • JavaScript es de tipado dinámico
  • Se pueden declarar sin valor inicial
  • Tienen ámbito de función, no de bloque
  • Se recomienda usar camelCase para los nombres
